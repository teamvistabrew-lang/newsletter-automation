Generate an AI newsletter from the provided data sources using the newsletter-gen skill.

Follow all steps in the SKILL.md file at .claude/skills/newsletter-gen/SKILL.md to:
1. Fetch content from the provided URLs
2. Pick the top 4 stories
3. Write a subject line and pre-header
4. Write story segments for each story
5. Write the newsletter intro
6. Write "The Shortlist" section
7. Assemble and output the complete newsletter

Arguments: $ARGUMENTS (URLs separated by spaces, or a path to a file containing URLs one per line)

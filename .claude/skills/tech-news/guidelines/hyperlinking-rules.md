# Hyperlinking Rules

## Core Principle

Every hyperlink must be verified, relevant, and extracted verbatim from the source material. Never fabricate, guess, modify, or complete URLs.

## Strategic Linking

1. Link specific companies, products, and technologies to their official source on first mention.
2. Link data, statistics, claims, or quotes to their precise origin.
3. For announcements, link back to the company's actual press release or blog post.

## Mandatory Verified Deep Links

- All hyperlinks MUST point to the exact, specific page supporting the assertion.
- Prioritize primary or most authoritative sources.
- **NO linking to generic homepages**.
- If the precise URL cannot be confidently identified, DO NOT include a link.

## URL Handling

- URLs must be **character-for-character copies** from the source material.
- **Strictly forbidden** from creating, guessing, modifying, or completing URLs.
- If a URL appears broken in the source, copy it exactly as-is.
- If no URL is available, omit the link and use plain text.

## Link Formatting

- Use brief, relevant anchor text (2-5 words).
- Limit to **1 link per paragraph or bullet point**.
- Do NOT format links as bold.
- Weave links naturally into the narrative.
- Do NOT link the same URL more than once per section.

## What NOT to Link

- Generic homepages or main site sections.
- Shopping/eCommerce pages.
- Video sites (prefer text-based sources).
- PDF files.
- Content you already have the full text for.

## What TO Link

- Press releases and official company blogs.
- Specific articles from credible publications.
- Tweets/X posts from verified accounts.
- Research reports (non-PDF web versions).

# Anthropic's Claude Can Now Search the Web in Real Time

**What's new?**

**AI Brief:** Anthropic [launched web search](https://anthropic.com/news/web-search) for Claude, allowing the AI assistant to pull live information during conversations. The feature is available to all Claude Pro and Team subscribers starting today.

**What matters?**

- This closes a **major gap** with ChatGPT and Gemini, both of which have had web access for months — Claude users no longer need to copy-paste context from browsers.
- The search feature is powered by a hybrid approach that combines [traditional web indexing](https://example.com/search-details) with Claude's reasoning to verify and synthesize results in real time.
- Early tests show Claude's web answers tend to be **more thorough** than competitors, often citing multiple sources and flagging conflicting information rather than picking one answer.

**Why it matters?**

Real-time information access transforms Claude from a knowledge-cutoff chatbot into a genuine research tool. Expect this to accelerate enterprise adoption where up-to-date data is non-negotiable.
